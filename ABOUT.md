# À propos de HylstDevToolBox

## Vision

HylstDevToolBox est une **boîte à outils universelle pour développeurs**, conçue par Geoffroy Streit (Hylst) pour centraliser les utilitaires du quotidien en une seule application fluide. 
Fini les dizaines d'onglets ouverts pour formater du JSON, tester une regex, ou générer des données factices. Tout est là, à portée de clic.

## Principes fondateurs

### 🔒 100% Hors-Ligne (PWA)
Aucune donnée ne quitte votre navigateur. Tous les traitements sont effectués côté client. L'application est installable (Progressive Web App) et fonctionne entièrement hors ligne. Idéal pour travailler dans les transports ou en zone blanche.

### 🎓 Pédagogique & Orienté Apprentissage
Créé dans le cadre de ma plateforme de support de cours ([hylst.fr](https://hylst.fr)), chaque outil intègre des explications contextuelles et des cheatsheets (aide-mémoires). L'objectif est d'aider les développeurs à comprendre, pas seulement à exécuter.

### 🇫🇷 Localisé et Humanisé
Une interface entièrement en français, pensée pour les développeurs francophones. Les termes techniques universels (JSON, API, UUID) sont évidemment conservés. 

### ⚡ Rapide et Extrêmement Pratique
- **Recherche globale instantaée**
- **Raccourcis clavier globaux** (Ctrl+Shift+Flèches, Ctrl+Shift+V pour l'historique du presse-papier)
- **Mode Présentation** pour se concentrer sur l'essentiel
- Construit avec React + Vite pour un chargement immédiat.

## Chiffres clés

- **120+ outils** fonctionnels
- **7 catégories** thématiques (Code, Design, Base de données, etc.)
- **0 tracking, 0 base de données externe** 
- **40+ composants** UI réutilisables (shadcn/ui)

## Stack technique

| Technologie | Rôle |
|-------------|------|
| **React 18** | Moteur de rendu UI |
| **TypeScript 5** | Typage statique robuste |
| **Vite 5** | Bundler ultra-rapide / Dev server |
| **Tailwind CSS** | Styling utilitaire et Design System |
| **shadcn/ui** | Composants UI accessibles et modulaires |
| **vite-plugin-pwa** | Gestion du Service Worker et de l'installation |
| **localForage / Dexie** | Gestion du stockage local (Favoris, Historique) |
| **Lucide React** | Icônes SVG légères |
